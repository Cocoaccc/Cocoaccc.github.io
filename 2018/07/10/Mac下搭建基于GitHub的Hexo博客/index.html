添加SSH Key到GitHub（解决每次都要输入GitHub账户名和密码）

  1. 执行以下命令，查询SSH Key是否存在
    $ ls -al ~/.ssh
    若存在如下文件::id_rsa.pub::或::id_dsa.pub::，则直接执行步骤3，否则继续下一步。

  2. 在本地创建SSH Key
    $ ssh-keygen -t rsa -C "your_email@youremail.com"
    注意将::your_email@example.com::换成你自己注册Github的邮箱地址，之后会要求确认路径和输入密码，一路回车即可。
    成功则会在~/下生成.ssh文件夹，cd进入该文件夹，打开id_rsa.pub，复制里面的key。

  3. 进入GitHub个人设置界面，添加SSH Key
